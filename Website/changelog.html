Changelog

  • Version 0.1.0 - 2/2/23
    • Game published to GitHub, with baseline functionality
    • Health Potions added
    • Several enemy types added
  • Version 0.1.1 - 2/3/23
    • Classes have been added. Class chosen randomly upon starting new game
      • Aries - Aries focuses on pure carnage, specing a high attack stat
      • Artemis - A jack-of-all-trades, Artemis starts with balanced stats
      • Aphrodite - Focused on sustainability, Aphrodite starts with a meaty HP stat.
    • WASD keybinds now work. For more information, see the controls page
